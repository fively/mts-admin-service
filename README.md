# My Tools Admin Service
Nest学习之旅

## Description
Nest学习之旅，个人学习项目，基于 Nestjs、 Fastify、 knex、 mysql等进行开发

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
